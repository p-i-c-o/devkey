Password Generator

In order to keep safe on the Internet, you need long and complex passwords, to keep malicious people out of your accounts.

Today, we will make a password generator that will do the hard work for you.

IMPORTS AND VARIABLES

In order for this script to work, we will need to import a package. The random package will give us the ability to generate random numbers, or selections. But in this project, we will only use the random.choice() command. We do so like this: 

import random

Now that we have that imported, we need to define what our password will be made up of, so we can simple define three variables: caps , lows , and special . The caps variable will contain the capital letters of the alphabet, the lows variable will contain the lowercase letters of the alphabet, and the special variable will contain special characters like *, #, !, and more. 

caps = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
lows = "abcdefghijklmnopqrstuvwxyz"
special = "!#$%&'()*+,-./:;<=>?@[\]^_`{|}~"

We want to make our script a little nicer to work with, so we will create a menu variable for both the length selection and the characters selection. You can use the one below: 

LengthMenu = """Password Generator:
(Choose length)
1. 8  Characters
2. 12 Characters
3. 16 Characters
4. 18 Characters"""
CharMenu = """
(Choose included characters)
1. Capital letters    (ex: ABC...)
2. Lowercase letters  (ex: abc...)

That’s it! Here is our final code block for this section.

INPUTS AND CHECKS

Now we have the back-end of the script set up, so we can now get to the actual fun part!

In order for the user to interact with the script, we will use the input() command to accept information. But before that, we will give the available options by printing out the necessary menu. 

print(LengthMenu)
LengthInput = input('[>]   ')

Then, to prevent the code from stalling, we will add a checker, to make sure the input is an acceptable request. 

if LengthInput == "1":
    length = 8
elif LengthInput == "2":
    length = 12
elif LengthInput == "3":
    length = 16
elif LengthInput == "4":
    length = 18
else:
    print("That's not an option, ending script now...")
    exit()

The first if statement checks if the input was 1 , the following check for 2 , 3 , and 4 . Each if statement then defines the input length as the desired input. If the input is not a proposed input, the script will run the else statement. It’ll then quit the script.

The next script excerpt asks for the character selection menu.

print(CharMenu)
CharInput = input('[>]   ')
for i in CharInput:
    if int(i) > 3:
        print("That's not an option, ending script now...")
        exit()

The excerpt first prints the menu, then asks for an input. The expected result can be any combination of 1 , 2 and 3 . So we want any input that isn’t 1 , 2 and 3 gone. We can simply do this using this loop: 

for i in CharInput:
    if int(i) > 3:
        print("That's not an option, ending script now...")
        exit()

It iterates through the input and checks if that character is bigger than 3, if it is, it tells us: That's not an option, ending script now... then quits the script.

Now that we have a suitable input for the character selection menu, we can now process it. 

CharList = ""
if "1" in CharInput:
    CharList += caps
if "2" in CharInput:
    CharList += lows
if "3" in CharInput:
    CharList += special

We first set the CharList variable to an empty string.

According to the character selection menu, 1 adds Capitals letters to the list, 2 adds Lowercase letters to the list, and 3 adds Special characters to the list. This snippet does exactly that

PASSWORD GENERATION AND RETURN 

FinalPassword = ""

for i in range(0, length):
    RandomLetter = random.choice(CharList)
    FinalPassword += RandomLetter

In this final snippet will generate the password from the desired length and character range.

We first define the FinalPassword variable as an empty string. We then loop a certain amount of times, which is defined by the length variable. Then, each time the loop happens, a random character is selected from the CharList variable, and is added to the FinalPassword variable.

There we go! We’ve finished the generation! Let’s now return the newly generated password to the user! 

print('Generating.')
print('Generating..')
print('Generating...')
print(f"""Final Password:
{FinalPassword}""")

We print a cool generating animation(you can change this to something cooler), and we print the FinalPassword variable!
